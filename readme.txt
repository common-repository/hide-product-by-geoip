Hide products by GeoIP is a easy-to-setup plugin for woocommerce. You can  hide products for different countries + hide products by currency. 

== Description ==
Hide products by GeoIP is a plugin that allows you to hide products dependengli of the current country of users. Also you can install [WOOCS](https://wordpress.org/plugins/woocommerce-currency-switcher/)  and hide the products by currency.

https://www.youtube.com/watch?v=_hfdB-niy4g

### The Plugin Features:

* Hide products in several countries
* Hide products by several currencies
* Easy to use for administrators


== Installation ==
* Download to your plugin directory or simply install via Wordpress admin interface.
* Activate.
* Use.

== Frequently Asked Questions ==

Q: Where can I see the plugin settings?
R: Woocommerce settings->Products->Display

Q: Can I show products by country instead of hiding them?
R: Yes! Check an option - "Reverse the query" 

Q: How can I hide products by currency?
R: First you need to install the plugin WOOCS - https://wordpress.org/plugins/woocommerce-currency-switcher/
